# 如何使用帕秋莉手册模组添加新的手册

>最后更新：2025-10-21 13:25

作为最出名的手册api,仅次于ftb任务的最优指引，帕秋莉手册的编写方式十分清晰，但相比ftb任务而言没那么简单

## 文件结构

使用数据包/帕秋莉自带的加载方式创建新手册的方式略微有一点点不同

### 数据包形式

数据包:`data/命名空间/patchouli_books/书ID/book.json`:手册格式
资源包:`assets/命名空间/patchouli_books/书ID/en_us/...`:类别，章节，模板

### 帕秋莉形式

帕秋莉形式则省去了命名空间的一步，在版本文件夹下`./patchouli_books`下的手册

### 总览

- patchouli_books
  - 书ID
    - book.json
    - en_us
      - entries
      - categories
      - templates
