# 倍增思想

>最后更新：2025-08-26 14:51

## 个人理解

**倍增**是二分的逆用，可以理解为对答案提前二分，组合预处理的数据

举例1:
使用砝码测量物体重量时，使用至少 $ceil(\log n)$ 个砝码，...32,16,8,4,2,1，通过组合得到原数

可以理解为从任意起点到终点的区间，从起点开始，长度为n的路径，例如 $\sum[l,r]$ ,通过组合预处理的结果，仅需计算 $\sum[l,l+2^i]+\sum[l+2^i,l+2^i+2^j]$ 的值  
仅需 $O(n\log n)$ 时间预处理，每次查询 $O(\log n)$ 时间，但无法修改  
空间复杂度 $O(n\log n)$

使用倍增思想的经典例子:[稀疏表/ST表](../数据结构/稀疏表.md)，线段树等
