# 使用数据包动态加载模组配置

>最后更新：2025-10-07 06:01

通过各种方式皆可得知，MC加载数据包使用了包`net.minecraft.server.packs.resources.SimpleJsonResourceReloadListener`
那么我们仅需根据其定义填充内容即可
通过查找源码实现，得知其实现`SimplePreparableReloadListener`抽象类
该抽象类提供了两个方法`prepare(ResourceManager,ProfilerFiller)`和`apply(T,ResourceManager,ProfilerFiller)`
其中`SimpleJsonResourceReloadListener`为我们实现了`prepare`方法，我们仅需继承自父类，并补充`apply`方法即可

```java

protected void apply(Map<ResourceLocation, JsonElement> resources,
            ResourceManager resourceManager, ProfilerFiller profiler) {


        for (Map.Entry<ResourceLocation, JsonElement> entry : resources.entrySet()) {
            entry.getKey(), entry.getValue()
        }
    }

```
