# 关于部分辅助模组的改动

>最后更新：2025-05-04 21:09

## 传送石碑

去除传送的经验消耗，提升其制作难度

`waystones-common.toml`

```toml
maximumBaseXpCost=0.0 # 最大经验消耗:0.0
generatedWaystonesUnbreakable=true # 自然生成的指路石不可破坏
globalWaystoneSetupRequiresCreativeMode=false # 非创造模式玩家依然可以创建全局指路石
frequency=0 # 禁用自然生成指路石
dimensionalWarpXpCost=1 # 跨维度传送消耗1级
globalWaystoneXpCostMultiplier = 2.147483647E9
warpStoneXpCostMultiplier = 2.147483647E9
waystoneXpCostMultiplier = 2.147483647E9
sharestoneXpCostMultiplier = 0.0
portstoneXpCostMultiplier = 2.147483647E9
warpPlateXpCostMultiplier = 0.0 # 除同族指路石和传送石盘，其余全部有2147483647倍率(无法跨维度传送)
```

删除了传送石碑和同族传送石碑的配方

## 背包

让背包不在成为高级容器
提升培养多个背包的价格，略微降低培养单个背包的价格

`sophisticatedbackpacks-common.toml`

```toml
chestLootEnabled = false # 不可在箱子战利品中获取背包
```

`sophisticatedbackpacks-server.toml`

```toml
[server.entityBackpackAdditions]
chance = 0.0 # 怪物不在携带背包生成
```
