# 洞穴世界的基本属性

| file           | key                             | 翻译               | 用途                                                  | 值                                 |
| -------------- | ------------------------------- | ------------------ | ----------------------------------------------------- | ---------------------------------- |
| dimension_type | ultrawarm                       | 过热               | 水桶是否能放水，湿海绵，岩浆，滴水石锥和泥巴          | false                              |
| dimension_type | natural                         | 自然               | 地狱门生成僵尸猪灵，床能否使用                        | true                               |
| dimension_type | coordinate_scale                | 坐标缩放           | 传送到该维度时的坐标缩放值                            | 1.0                                |
| dimension_type | has_skylight                    | 存在天空光照       | 该维度是否有天空光照                                  | true                               |
| dimension_type | has_ceiling                     | 存在基岩天花板     | 影响地图的行为,仅为逻辑上是否有天花板                 | true                               |
| dimension_type | ambient_light                   | 环境光照           | 该值越高，渲染得越亮,0.5时0和15光照等级渲染的方块无异 | 0.1                                |
| dimension_type | fixed_time                      | 固定时间           | （可选）该维度的昼夜时间将会固定在这个指定值上        | /                                  |
| dimension_type | monster_spawn_light_level       | 怪物生成最大光照   | 怪物生成位置的最大光照(包含天空光照)                  | 0                                  |
| dimension_type | monster_spawn_block_light_limit | 怪物生成最大亮度   | 怪物生成位置的最大方块光照                            | 0                                  |
| dimension_type | piglin_safe                     | 猪灵安全           | 猪灵、猪灵蛮兵和疣猪兽是否不会僵尸化                  | false                              |
| dimension_type | bed_works                       | 床能否工作         | 玩家试图使用床时，其是否不会爆炸                      | true                               |
| dimension_type | respawn_anchor_works            | 重生锚是否工作     | 玩家试图使用重生锚时，其是否不会爆炸                  | false                              |
| dimension_type | has_raids                       | 存在袭击           | 带有不祥之兆的玩家是否可以触发袭击                    | true                               |
| dimension_type | logical_height                  | 逻辑高度           | 玩家使用紫颂果或下界传送门可以到达的总高度            | 512                                |
| dimension_type | min_y                           | 最低高度           | 该维度中可以存在方块的最低高度                        | -512                               |
| dimension_type | height                          | 总高度             | 该维度中可以存在方块的总高度                          | 512                                |
| dimension_type | infiniburn                      | 可永久燃烧方块     | 该维度中火可以在哪些方块上永久燃烧                    | "\#minecraft:infiniburn_overworld" |
| dimension_type | effects                         |                    | 该维度的天空效果                                      | "minecraft:overworld"              |
| noise_settings | sea_level                       | 海平面             | 海平面高度                                            | -481(-512+32-1)                    |
| noise_settings | disable_mob_generation          | 禁用生物随地形生成 | 禁用生物随地形生成                                    | false                              |
| noise_settings | ore_veins_enabled               | 矿脉生成           | 原版铜铁矿脉生成                                      | false                              |
| noise_settings | aquifers_enabled                | 含水层生成         | 含水层生成                                            | false                              |
| noise_settings | legacy_random_source            | 旧随机数生成器     | 使用旧随机数生成器                                    | false                              |
| noise_settings | default_block                   | 默认方块           | 默认方块                                              | 深板岩                             |
| noise_settings | default_fluid                   | 默认流体           | 默认流体                                              | 岩浆                               |
